# 教程草稿

这个目录用于存放从 `docs/process-log/` 整理出来的正式教程章节。

教程目标是：

> 基于 ARE/Gaia2-style environments 构建工业级 agentic RL infrastructure。

计划章节：

1. 从 benchmark 到 training environment。
2. 设计可验证的 personal-assistant scenarios。
3. 收集 rollout traces。
4. 构建 action-level rewards。
5. 生成 teacher trajectories。
6. 面向 tool-use agents 的 SFT。
7. 面向 dynamic agent tasks 的 RLVR/GRPO。
8. 评测、ablation 和 failure analysis。
